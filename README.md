# Q-RIOUS Quiz Management System

An interactive web-based quiz management system designed for efficient academic assessments. This project features a "soft-pink glass-vibe" aesthetic for a modern user experience.

## 🚀 Features
* **Admin Module:** Manage users and monitor system activity.
* **Instructor Module:** Create, edit, and manage quiz content.
* **Participant Module:** Seamless quiz-taking interface with real-time scoring.
* **Database Management:** Integrated with SQL for robust data storage.

## 🛠️ Technologies Used
* **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript
* **Backend:** PHP
* **Database:** MySQL / SQL
* **Tools:** Git, GitHub, ByetHost

## 🔧 Installation Guide
1. Clone this repository.
2. Import `b8_41189529_Online_Quiz.sql` into your PHPMyAdmin.
3. Update `dbconnect.php` with your local or host credentials.
4. Run the project via XAMPP or live server.
